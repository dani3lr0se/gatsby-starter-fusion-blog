## Getting started

1. Create a Gatsby site

 ```shell
    gatsby new your-name-here https://github.com/gatsbyjs/gatsby-starter-fusion-blog
 ```
  
2. Start developing

```shell
    cd your-name-here
    gatsby develop
 ```

3. Open and edit the source code

Your site is now running at `http://localhost:8000`!
